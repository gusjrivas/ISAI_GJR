# Prompt

Crea un archivo `index.html` single-page que tenga el CSS y JavaScript incrustado, sin necesitar instalar ninguna dependencia.

La página debe mostrar inicialmente la fecha actual para que una persona pueda elegir el día de su nacimiento.

Cuando el usuario presione sobre la fecha, debe abrirse un modal fullscreen con una representación visual de la Tierra orbitando alrededor del Sol.

La interacción principal debe ser completamente visual e inmersiva:

- El usuario deberá girar la Tierra en sentido contrario a las agujas del reloj para viajar hacia atrás en el tiempo.
- A medida que la Tierra gira, la fecha deberá retroceder dinámicamente día por día.
- Cada vuelta completa de la órbita debe representar aproximadamente un año.
- La animación debe ser fluida y moderna.
- La Tierra debe verse orbitando realmente alrededor del Sol.
- El usuario podrá arrastrar la órbita con mouse o touch.
- También debe funcionar con teclado.

Cuando el usuario deje de girar y presione Enter:

- La fecha seleccionada deberá establecerse como fecha de nacimiento.
- El modal deberá cerrarse.
- La fecha elegida deberá mostrarse en el input principal.

Requisitos técnicos:

- Todo debe estar contenido en un único archivo `index.html`.
- No usar frameworks.
- No usar librerías externas.
- No usar dependencias.
- Solo HTML, CSS y JavaScript vanilla.
- Debe ser responsive.
- Debe tener un diseño visual moderno tipo space UI / sci-fi.
- Debe incluir efectos visuales suaves, sombras, gradientes y animaciones.
- El fondo debe representar el espacio.
- Debe verse profesional y listo para producción.

Extras deseados:

- Mostrar la cantidad de días recorridos hacia atrás.
- Permitir cerrar el modal con ESC.
- Permitir confirmar la fecha con Enter.
- Incluir pequeñas instrucciones visuales para el usuario.
- Evitar el uso de controles tradicionales de calendario.

La experiencia debe sentirse como “viajar en el tiempo hasta la fecha de nacimiento”.
